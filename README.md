# GitHub Profile Analyzer

A JavaScript application that fetches and analyzes GitHub user profile data using the GitHub REST API.

## 🚀 Features

- Fetch GitHub user information
- Display profile name and username
- Show bio
- Show followers and following count
- Show public repository count
- Display profile URL
- Handle invalid usernames
- Handle API errors gracefully

---

## 🛠️ Technologies Used

- JavaScript
- Node.js
- GitHub REST API
- Async/Await
- Fetch API

## 📌 How It Works

1. User enters a GitHub username.
2. The application sends a request to the GitHub API.
3. Profile information is retrieved.
4. The data is formatted and displayed.
5. Errors are handled if the username does not exist or the request fails.
